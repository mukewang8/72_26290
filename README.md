<h3>课程介绍：</h3>
<p>CLinux短期班的项目是编译器 + 网盘项目，内容包括常用数据结构和算法，编译原理，操作系统原理实战，内核原理解析，计算机网络实战等。如果原有没有C语言基础，或者C语言学的不怎么好，没有编程基础，建议报C短期班，因为C短期班难度低于C++短期班。短期班是会提升编程能力和项目经验的，是锦上添花。如果是学历背景不错，或者科班的，春招是很不错的。</p>
